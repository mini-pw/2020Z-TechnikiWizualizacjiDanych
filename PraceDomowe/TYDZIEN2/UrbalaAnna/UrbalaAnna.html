Dane rdowe

BIG InfoMonitor - Finansowa sytuacja byaby lepsza, gdyby nie rozrzutno i ycie ponad stan

Dane pokazuj, jak cz dochodw ludzie w rnym wieku trac przez rozrzutno. Mamy podan frakcj osb z danej kategorii wiekowej, ktre dany procent dochodu przeznaczaj na zbyt drogie rzeczy. Wybraam wykres procentowy, gdy dobrze wida, jak ukadaj si procenty, m.in. mona bez problemu okreli, wrd ktrych grup najmniej ludzi wydaje ponad 5% albo nic.

Wczytanie danych

Dane do pliku dane.csv musiaam rcznie przepisa ze strony.

data <- read.csv(
  'dane.csv',
  header = TRUE,
  sep = ",")

Dodatkowa kolumna do ustawiania pozycji etykiet

data <- data %>%
  group_by(age) %>%
  arrange(age, factor(proc, levels = c("ponad 20%", "10-20%", "5-10%", "3-5%", "do 3%", "nic"))) %>%
  mutate(lab_ypos = cumsum(value) - 0.5 * value)

Paleta kolorw

palette <- c("#FF9999", "#FF6666", "#FF0000", "#CC0000", "#990000", "#660000")

Generowanie wykresu

ggplot(data,
             aes(
               y = value,
               x = age,
               fill = factor(
                 proc,
                 levels = # sortowanie po frakcji
                   c("nic", "do 3%", "3-5%", "5-10%", "10-20%", "ponad 20%")))) + 
  geom_bar(stat = "identity") +
  scale_fill_manual(values = palette) + # ustawienie palety
  labs(
    title = "Jak cz miesicznych dochodw tracisz, kupujc zbyt drogie rzeczy?",
    fill = "Frakcja dochodw",
    x = "Wiek",
    y = "Frakcja osb") +
  scale_y_continuous(labels = dollar_format(suffix = "%", prefix = "")) + # dodanie prefiksow do skali y
  geom_text( # dodanie wartosci procentowych w srodku pol
    aes(y = lab_ypos, label = paste(value, "%", sep = ""), group = age),
    color = "white")
